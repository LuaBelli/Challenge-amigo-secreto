# 🎀👓 Challenge Amigo Secreto 👊✨

Bem-vindo(a) ao meu projeto Amigo Secreto!
Um projeto feito com HTML, CSS e JavaScript, onde você pode cadastrar seus amigos e sortear quem será o sortudo(a).

Sorteio bom é aquele com muito aprendizado e estudo em lógica de programação, glamour, atitude e um ippon na hora certa! (piadinha de judoca)

---

🚀 Funcionalidades

➕ Adicionar amigos em uma lista visível na tela

✅ Validação para evitar nomes em branco

👀 Visualização dinâmica da lista de participantes

🎲 Sorteio aleatório de um amigo secreto

🎉 Efeito de confete animado em SVG para celebrar o resultado

---

🛠️ Tecnologias utilizadas

HTML5 → Estrutura

CSS3 → Estilização e responsividade

JavaScript → Toda a lógica de programação do sorteio e interatividade

---

📂 Como usar

1. git clone https://github.com/LuaBelli/Challenge-amigo-secreto


2. Abra o arquivo index.html no navegador.


3. Adicione nomes, clique em Sortear Amigo e divirta-se!
